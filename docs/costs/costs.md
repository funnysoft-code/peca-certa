## How to refresh

```bash
# Prerequisites: cloud CLI auth, wrangler login (CF workers)
# xAI costs accumulate automatically from inference responses.
php artisan costs:update
php artisan costs:update --dry-run
```

Config: `config/costs.php`. Machine history: `docs/costs/costs.json`.
xAI ledger: `storage/app/private/costs/xai-usage.jsonl` (Inference API `cost_in_usd_ticks`).
xAI key-scoped totals: set `XAI_MANAGEMENT_KEY` + `XAI_TEAM_ID` (optional `XAI_API_KEY_ID` / `XAI_API_KEY_NAME=peca-certa`).
